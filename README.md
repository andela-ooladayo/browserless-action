# Browserless GitHub Action

This [GitHub Action](https://github.com/features/actions) sets up a Browserless Chrome.

# Usage

Basic:
```yaml
steps:
- uses: andela-ooladayo/browserless-action@v1.0
   with:
    port: 3000
    version: latest
```
